# realwds's blog

🏡 Butterfly hexo 主题魔改版本

访问地址：https://realwds.com

## Usage

``` sh
# 进入文件目录
cd source

# 安装
# 建议不要直接使用 cnpm 安装依赖，会有各种诡异的 bug。可以通过如下操作解决 npm 下载速度慢的问题
pnpm install --registry=https://mirrors.huaweicloud.com/repository/npm/

# 启动
pnpm dev

# 新建文件夹
hexo new page <文件夹名称>

# 新建文件
hexo new <文章标题>

# 部署
pnpm build

# 提交
pnpm upload <提交信息说明>
```
