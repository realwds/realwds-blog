## 移除首尾空格

``` js
// 方式 1 正则表达式
str.replace(/(^\s*)|(\s*$)/g, "")

// 方式 2 trim()
str.trim()
```

## 空格批量替换

``` js
// 正则替换一个及以上空格为逗号
str.replace(/\s+/g,",")
```

## 批量替换函数

``` js
// 批量替换函数
function replaceStr(str) {
  return str.replace(/(^\s*)|(\s*$)/g, "").replace(/\s+/g, ",")
}

// 样例 demo
replaceStr('1,2  3  ')    // 1,2,3
replaceStr(' 1 2 3  ')    // 1,2,3
replaceStr(' 1  2 3  ')   // 1,2,3
```
