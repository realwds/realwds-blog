## 关机/重启/注销

| 作⽤            | 命令                |
|---------------|-------------------|
| **即刻关机**          | `shutdown -h now`   |
| **10分钟后关机**       | `shutdown -h 10`    |

| **10分钟之后重启**      | `shutdown -r 10`    |

| **buff数据同步到磁盘**   | `sync`              |

| **更新已安装软件包**    | `apt-get upgrade`           |
| **清理缓存**        | `apt-get clean`             |
